# eSport Manufaktur - Landing Page

## Overview
A React-based landing page for eSport Manufaktur, featuring a modern Linear-style design. Built with React 19, TypeScript, Vite, and Tailwind CSS.

## Project Structure
- `App.tsx` - Main application component
- `index.tsx` - Entry point
- `index.html` - HTML template
- `index.css` - Global styles
- `components/` - React components
- `vite.config.ts` - Vite configuration

## Tech Stack
- React 19
- TypeScript
- Vite 6
- Tailwind CSS (via CDN)
- Framer Motion (animations)
- Lucide React (icons)

## Development
Run `npm run dev` to start the development server on port 5000.

## Build
Run `npm run build` to create a production build in the `dist` directory.

## Deployment
Configured for static deployment. The build outputs to `dist/` directory.
